# Стоп-лист донорів (StopList Tool)

Веб-інструмент для ведення стоп-листа донорів: імпорт ТЗ, дедуплікація,
статистика, експорт у Excel та робота з брендами. Бекенд — Flask (Python).

## Можливості

- **Імпорт ТЗ** — завантаження нових технічних завдань для розширення стоп-листа
- **Стоп-лист** — перегляд унікальних пар «акцептор — донор»
- **Статистика** — кількість акцепторів, донорів, пар, дублікатів
- **Експорт** — вивантаження стоп-листа у файли Excel
- **Бренди** — облік брендів

## Технології

- Python 3 + Flask
- SQLite (локальна база `stoplist.db`)
- openpyxl / XlsxWriter — генерація Excel
- Bootstrap 5 — інтерфейс

## Запуск

```bash
# 1. Створити віртуальне середовище та встановити залежності
python -m venv venv
venv\Scripts\activate        # Windows
pip install -r requirements.txt

# 2. Запустити застосунок
python app.py
```

Або просто запустити `start.bat`.

Після старту відкрийте у браузері: http://localhost:5555

## Структура проєкту

```
app.py            # точка входу Flask, маршрути
config.py         # шляхи, порт, налаштування
db/               # схема та SQL-запити (queries.py, schema.py)
services/         # парсер ТЗ, експорт, інтеграція з Google Sheets
templates/        # HTML-шаблони (Jinja2 + Bootstrap)
static/           # app.js, style.css
scan_brands.py    # утиліта сканування брендів
```

## Примітка про дані

Файл бази `stoplist.db` та згенеровані `exports/*.xlsx` **не зберігаються в
репозиторії** (див. `.gitignore`) через великий розмір. База створюється
локально при першому запуску.
